# AI_QUANT_LAB_V4

## Lancer tous les tests de modules

Depuis la racine du projet, exécute :

```
python AI_QUANT_LAB_V4/run_all_module_tests.py
```

Ce script lance automatiquement les tests pour :
- DNA Evolution & Selection
- Market Regime AI

Les résultats de chaque module s’affichent dans la console.

## Structure du projet

- data_intelligence
- research_ai
- strategy_ecosystem
- simulation_lab
- risk_intelligence
- portfolio_ai
- execution_system
- supervision
- infrastructure

Chaque dossier contient ses propres README et sous-modules.

---

Pour ajouter un nouveau test, ajoute simplement le chemin du script dans la liste `MODULE_TESTS` du fichier `run_all_module_tests.py`.
