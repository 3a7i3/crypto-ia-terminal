## How to Restore a Module

If a module is needed again:

1. **Check git history**:
   ```bash
   git show HEAD:quant_hedge_ai/agents/research/feature_engineer.py
   ```

2. **Restore the file**:
   ```bash
   git checkout HEAD -- quant_hedge_ai/agents/research/
   ```

3. **Or restore from this archive**:
   ```bash
   cp -r quant_hedge_ai/_legacy/archived_modules/research quant_hedge_ai/agents/
   ```

4. **Write tests** before reintegration (required for production use)
