# ☁️ Démo d’intégration cloud – Streamlit Cloud

## 1. Prérequis
- Compte GitHub
- Compte Streamlit Cloud (https://streamlit.io/cloud)

## 2. Étapes
1. Pousser le dossier du projet sur GitHub (public ou privé)
2. Aller sur https://streamlit.io/cloud et connecter votre compte GitHub
3. Sélectionner le repo, choisir le fichier à lancer (ex : `evolution_3d_view.py`)
4. Configurer les variables d’environnement si besoin (onglet « Advanced settings »)
5. Lancer le déploiement

## 3. Résultat
- Un lien public sécurisé est généré (ex : https://cloud.streamlit.io/your-app)
- Les utilisateurs accèdent au dashboard sans installation locale

## 4. Conseils
- Pour un usage privé, activer l’authentification Streamlit Cloud
- Pour d’autres clouds (Azure, AWS, GCP), voir la doc Streamlit : https://docs.streamlit.io/streamlit-community-cloud/deploy
