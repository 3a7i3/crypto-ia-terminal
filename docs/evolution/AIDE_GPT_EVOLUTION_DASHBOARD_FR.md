# 🤖 Aide interactive – Assistant GPT pour l’écosystème évolutif

## Exemples de questions à poser à l’assistant :

- « Comment lancer la simulation évolutive ? »
- « Comment filtrer les stratégies par espèce et fitness ? »
- « Comment utiliser l’AutoML pour optimiser TP et SL ? »
- « Comment exporter les résultats pour Excel ? »
- « Comment interpréter la heatmap ou le clustering ? »
- « Comment comparer les mondes entre eux ? »
- « Comment relancer la simulation si j’ai une erreur ? »
- « Comment utiliser les archives ? »

## Bonnes pratiques pour interroger l’assistant :
- Soyez précis sur le contexte (dashboard, simulation, export…)
- Précisez le type d’analyse ou d’optimisation souhaité
- Demandez des exemples de code ou de manipulation si besoin
